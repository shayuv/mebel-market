# ТЗ: Кликабельный прототип мебельного интернет-магазина

**Версия:** 1.0  
**Дата:** 01.04.2026  
**Подготовил:** azat.team  
**Бюджет часов:** до 200 ч  
**Цель:** Пресейл-демо — кликабельный прототип на Next.js с моковыми данными, mobile-first, современный UI

---

## 1. Референсы и ключевые паттерны

**Hoff.ru** — крупнейший мебельный e-commerce в РФ:
- Фиксированное левое меню каталога с раскрывающимися подкатегориями
- Мощная система фильтров по вкладкам (цена, материал, цвет, размер, бренд)
- Мини-карточки товаров с рейтингом, ценой, бейджами «Хит», «Скидка»
- Карточка товара: галерея, видео, фото от покупателей, отзывы, похожие товары, «купить в 1 клик»
- Бонусная программа, сторис-блок на главной, готовые интерьерные решения
- Геолокация города влияет на цены/наличие

**Nonton.ru** — онлайн-гипермаркет мебели, бюджетный сегмент:
- Топлайн-баннер с акцией (скидки до 70%)
- Мега-меню с иконками категорий (Диваны и кресла, Шкафы, Кровати, Кухни, Столы, Детская, Ванная, Прихожая, Бытовая техника)
- Простая навигация: хлебные крошки, подкатегории-ссылки
- Блок «Почему выгодно у нас» (прямые поставки, гарантия, сборка)
- Отзывы покупателей на главной

## 3. Детальное описание экранов

### 3.1 Главная страница

**Header (sticky, все экраны):**
- Топлайн: выбор города + телефон + ссылки (Доставка, Оплата, Магазины)
- Основной ряд: логотип, кнопка «Каталог» (бургер на мобилке), поиск (расширяющийся input), иконки (избранное + count, сравнение + count, корзина + count + мини-сумма), кнопка «Войти»
- На мобилке: sticky bottom-bar (Главная, Каталог, Корзина, Избранное, Профиль)

**Hero-секция:**
- Полноширинный слайдер (3–5 баннеров), autoplay, swipe на мобилке
- CTA-кнопка на каждом слайде

**Категории:**
- Горизонтальный скролл иконок категорий (8–12 шт) с названиями
- На десктопе — сетка 2 ряда × 6 колонок

**Промо-блок:**
- 2–4 карточки акций (баннер-плитка), ссылки на подборки

**Хиты продаж / Новинки:**
- Горизонтальный карусель товаров (ProductCard компонент)
- Переключатель табов: «Хиты» / «Новинки» / «Скидки»

**Готовые интерьерные решения (отличие от конкурентов):**
- Карусель фото интерьеров с тегами товаров (как Pinterest shoppable pins)
- Клик по тегу → переход на карточку товара

**Преимущества:**
- 4 иконки + текст (Бесплатная доставка от X ₽, Гарантия, Рассрочка 0%, Сборка)

**Отзывы:**
- Карусель последних отзывов с рейтингом и фото

**SEO-блок:**
- Текстовый блок с описанием магазина (скрыт за «Показать ещё»)

**Footer:**
- 4 колонки: Каталог, Покупателям, О компании, Контакты
- Соцсети, copyright, ссылки на мобильное приложение (заглушки)

---

### 3.2 Каталог (категория)

**Layout:**
- Desktop: sidebar (фильтры) 280px + контент
- Mobile: фильтры в выезжающем drawer снизу

**Хлебные крошки:**
- Главная → Категория → Подкатегория

**Подкатегории:**
- Горизонтальные чипсы / плитки с иконками (прямые диваны, угловые, кресла…)

**Панель сортировки + отображения:**
- Сортировка: Популярные / Дешёвые / Дорогие / Новинки / По рейтингу
- Переключатель: сетка (2/3/4 колонки desktop, 1/2 мобилка) / список
- Счётчик: «Найдено 248 товаров»

**Фильтры (sidebar / drawer):**
- Цена: двойной ползунок (range slider) + инпуты min/max
- Бренд/Производитель: чекбоксы с поиском внутри
- Материал: чекбоксы (ЛДСП, МДФ, массив, экокожа, ткань…)
- Цвет: палитра цветных кружков
- Размер: инпуты ШxВxГ + пресеты
- В наличии: тоггл
- Со скидкой: тоггл
- Кнопки: «Применить» + «Сбросить всё»
- Тег активного фильтра → кнопка удаления тега

**ProductCard (карточка в каталоге):**
- Фото (hover → второе фото или стрелки на мобилке)
- Бейджи: «Хит», «−30%», «Новинка» (лейблы в углу)
- Название товара (2 строки max)
- Рейтинг (звёздочки + кол-во отзывов)
- Цена: старая (зачёркнутая) + новая (жирная, акцентный цвет)
- Кнопка «В корзину» (на hover / всегда на мобилке)
- Иконки: избранное ♡, сравнение ⇆

**Пагинация:**
- «Показать ещё» (infinite scroll на мобилке) + нумерация страниц на десктопе

---

### 3.3 Карточка товара

**Галерея:**
- Desktop: основное фото (zoom on hover) + вертикальная лента превью слева
- Mobile: горизонтальный swipe + точки пагинации
- Поддержка видео-слайда (иконка play)
- Фото от покупателей (мини-лента)

**Информация:**
- Название, артикул, рейтинг со ссылкой на отзывы
- Цена: старая/новая, бейдж скидки, «Экономия X ₽»
- Варианты: переключение цвета (кружки-свотчи) + размера
- Статус: «В наличии» (зелёный) / «Под заказ, 7 дней»
- Кнопки: «В корзину» (primary) + «Купить в 1 клик» (secondary)
- Иконки: избранное, сравнение, поделиться
- Краткие характеристики (3–5 ключевых: ШxВxГ, материал, цвет, вес)
- «Рассрочка от X ₽/мес» — информационный блок

**Табы (ниже fold):**
- Описание (текст + доп. фото)
- Характеристики (таблица key-value)
- Отзывы (список с рейтингом, фото, ответ магазина; кнопка «Написать отзыв»)
- Доставка и оплата (условия по выбранному городу)

**Сопутствующие блоки:**
- «С этим товаром покупают» — карусель
- «Похожие товары» — карусель
- «Недавно просмотренные» — карусель

---

### 3.4 Корзина

**Список товаров:**
- Фото-превью + название + артикул
- Выбранный вариант (цвет/размер)
- Цена за единицу
- Количество: −/+/инпут
- Сумма по позиции
- Кнопка удалить (×)
- «Перенести в избранное»

**Сайдбар итогов (sticky на десктопе):**
- Количество товаров
- Сумма без скидки
- Скидка
- Итого
- Промокод (инпут + «Применить»)
- Кнопка «Оформить заказ» (primary, CTA)
- Иконки оплаты (Visa, MС, Mir, СБП)

**Пустая корзина:** иллюстрация + «Ваша корзина пуста» + кнопка «Перейти в каталог»

---

### 3.5 Оформление заказа (Checkout)

**Шаги (прогресс-бар сверху):**
1. Контактные данные (имя, телефон, email)
2. Доставка: табы «Доставка курьером» / «Самовывоз» → адрес + дата/время слот
3. Оплата: «Онлайн картой» / «При получении» / «Рассрочка»
4. Подтверждение: сводка заказа + кнопка «Подтвердить»

**Мобилка:** аккордеон-стиль, по шагу за раз

---

### 3.6 Поиск

- Инпут с live-suggest (по мере ввода: категории + товары + подсказки)
- Результаты: та же сетка что и каталог + фильтры
- Пустой стейт: «Ничего не найдено» + популярные запросы + рекомендации

---

### 3.7 Избранное

- Сетка ProductCard без фильтров
- Пустой стейт: «Добавьте товары в избранное» + кнопка «Перейти в каталог»

---

### 3.8 Сравнение товаров

- Desktop: горизонтальная таблица (товар — колонка, характеристика — строка)
- Mobile: горизонтальный свайп карточек + «Показать только различия»
- Максимум 4 товара

## 6. Мобильная адаптация (ключевые решения)

- **Mobile-first** подход: всё проектируется сначала для 375px
- **Bottom navigation bar** вместо бургер-меню (Главная, Каталог, Корзина, Избранное, Профиль)
- **Фильтры** — bottom sheet (drawer снизу, как в Яндекс.Маркете)
- **Галерея товара** — fullscreen swipe с pinch-to-zoom
- **Корзина** — sticky footer с итоговой суммой и CTA
- **Чекаут** — one-step per screen, крупные тач-таргеты (мин. 48px)
- **Каталог** — 2 колонки, бесконечный скролл
- **Поиск** — полноэкранный overlay при тапе

---

## 7. Интерактивные фишки для wow-эффекта на пресейле

1. **Shoppable интерьеры** — клик по точке на фото интерьера → попап товара
2. **Quick View** — модал с основной информацией из каталога без перехода
3. **Sticky «В корзину»** на карточке товара (мобилка)
4. **Анимация добавления в корзину** — товар «улетает» в иконку корзины
5. **Live-поиск** с автокомплитом и карточками товаров прямо в dropdown
6. **Сравнение** — swipe-карточки на мобилке
7. **Переключение цвета** — фото товара меняется при выборе свотча
8. **Skeleton loading** — красивый лоадинг вместо спиннеров
9. **Микро-анимации** — hover-эффекты, transitions на кнопках
