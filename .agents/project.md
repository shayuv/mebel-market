# МЕБЕЛЬ.маркет — Кликабельный прототип

## Цель
Пресейл-демо мебельного интернет-магазина. Кликабельный прототип с моковыми данными.
**НЕ** готовый магазин — без бэкенда, авторизации, оплаты.

## Стек
- **Framework:** Next.js 16 (App Router)
- **Styles:** Tailwind CSS 4 + shadcn/ui
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Sliders:** Swiper.js
- **Data:** Моковый JSON (faker, 50–100 товаров)
- **Images:** Unsplash placeholders
- **Deploy:** Vercel

## Референсы
- Hoff.ru — мега-меню, фильтры, карточки товаров
- Nonton.ru — промо-баннеры, простая навигация
