# Sprint 1 — Каталог + Карточка товара

**Цель:** Полный флоу: каталог → фильтры → карточка товара
**Приоритет:** P0
**Агент:** Build

## Задачи

### 1. Каталог — верхний уровень (6ч)
- [ ] Страница /catalog — сетка подкатегорий с иконками
- [ ] Хлебные крошки (Breadcrumbs)
- [ ] Чипсы подкатегорий

### 2. Каталог — подкатегория с фильтрами (12ч)
- [ ] Сетка товаров (2/3/4 колонки, переключатель)
- [ ] SortBar (сортировка + вид + счётчик)
- [ ] FilterSidebar (desktop): цена (range slider), бренд, материал, цвет, размер, тогглы
- [ ] FilterDrawer (mobile): bottom sheet с фильтрами
- [ ] Активные фильтры — теги с удалением
- [ ] Пагинация / "Показать ещё"

### 3. Карточка товара (12ч)
- [ ] ProductGallery: основное фото + превью (desktop), swipe (mobile)
- [ ] Информация: цена, варианты (цвет/размер), статус, кнопки
- [ ] ProductTabs: описание, характеристики, отзывы, доставка
- [ ] "С этим покупают" + "Похожие товары" — карусели
- [ ] Sticky "В корзину" на мобилке

### 4. Поиск (4ч)
- [ ] Live-suggest dropdown (категории + товары)
- [ ] Страница /search с результатами
- [ ] Пустой стейт
