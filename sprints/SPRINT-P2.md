# Sprint 2 — Корзина + Чекаут + Остальные страницы

**Цель:** Полный пользовательский флоу до оформления заказа + второстепенные страницы
**Приоритет:** P0-P2
**Агент:** Build

## Задачи

### 1. Корзина (6ч)
- [ ] Список товаров (фото, название, количество ±, удалить)
- [ ] CartSummary (sticky sidebar: итого, промокод, CTA)
- [ ] Пустая корзина (иллюстрация + CTA)
- [ ] Глобальный стейт корзины (React Context или Zustand)

- [ ] Step 2: доставка (курьер/самовывоз)
- [ ] Step 3: оплата (карта/при получении/рассрочка)
- [ ] Step 4: подтверждение (сводка)
- [ ] Мобилка: аккордеон по шагам

### 3. Избранное (2ч)
- [ ] Сетка ProductCard
- [ ] Пустой стейт

### 4. Сравнение (4ч)
- [ ] Desktop: таблица (товар-колонка)
- [ ] Mobile: swipe-карточки
- [ ] "Показать только различия"

### 5. Информационные страницы (3ч)
- [ ] Доставка, Оплата, Гарантии — шаблон
